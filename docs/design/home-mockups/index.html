Home Page Redesign — Three Directions

Three distinct philosophies for the BharatAgriLens home screen. All three use the existing Krishi Sahayak palette (leaf, saffron, soil, ink) so the rest of the app doesn't need re-skinning. Open each in a new tab to interact with it at full mobile width.

Tap the frame title to open in a new tab.

Direction A

My Field Today 🌱 →

One big crop card. Everything else orbits around it.
Audience
Daily engaged farmers
Effort
~6 hours

✅ Strengths

• Visual-first hero answers "what's next?" in 1 glance
• Uses Trigger #2 treatment-reminder data we're already capturing
• Photo-rich recent scans match how farmers think

⚠ Trade-offs

• Empty state needs a "Take your first scan" prompt for new users
• Needs crop list to be populated (data-dependent)
Direction B

Mission Control 📊 →

Premium dashboard. In command of your farm.
Audience
Power users, agronomists
Effort
~8 hours

✅ Strengths

• Most premium-feeling design
• Pre-figures Trigger #3 outbreak data ("In your area")
• Sparkline stats add data depth

⚠ Trade-offs

• Dark hero is a departure from the rest of the app
• Empty stats look thin until data accumulates
• Highest cognitive load
Direction C

Action First ⚡ →

Cleanest, simplest. Three taps to anywhere.
Audience
Occasional / first-time
Effort
~3 hours

✅ Strengths

• Easiest to learn — large tiles, noun labels
• Empty state still looks intentional
• Search bar enables quick text questions

⚠ Trade-offs

• No personalisation signal
• No "next action today" urgency
• Closest to existing flat layout

💡 Recommendation

Pick A "My Field Today" is the right fit for your existing engaged user base. The hero card answers "what's happening with my crops right now?" in one glance, uses the treatment-reminder data we just built (Trigger #2), and visually celebrates the photo-of-the-leaf workflow that's the app's USP.

Specific pitch: build Direction A as the base, then borrow the "In your area" outbreak panel from Direction B and slot it in just below the hero card. Best of both — daily-user-first hero plus the unique hyperlocal signal no competitor has.

Direction C is right if you're optimising for first-impression conversion (KVK event downloads), but your existing daily users don't need to "discover the app" — they need help with their crops. Direction B is right for v2 once outbreak + treatment data has accumulated enough to make the panels feel inhabited.
